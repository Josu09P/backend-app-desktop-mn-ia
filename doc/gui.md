# ⚙️ Guía de instalación y ejecución del backend Flask

Este documento explica paso a paso cómo instalar, configurar y ejecutar correctamente un **backend en Python (Flask)** para cualquier entorno (Windows o Linux).
---
## CREAR ENTORNO VENV en linux
- python3 -m venv venv
- source venv/bin/activate

### en windos
- python -m venv venv
- venv\Scripts\activate

## EJECUTAR EN EL TERMINAL PARA INSTALAR LAS LIBRERIAS
- pip install -r requirements.txt

## EJECUTAR PARA CORRER EL SERVER (BACKEND)
- python main.py
